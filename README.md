# StatsExtractor

ワールドに入ったことのある全プレイヤーの統計情報をCSVにエクスポートします。  
`plugins/StatsExtractor`に名前、UUID、所属チーム名とともに統計データが出力されるため、Excelやスプレッドシートで加工して使用してください。

## 使い方

`/stats-players` コマンドを実行してください。  
あとは自動で収集されます。(50人の統計収集におおよそ3分くらいかかります)
